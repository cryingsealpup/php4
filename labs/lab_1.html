Модуль 1. Продвинутые возможности

Лабораторная 1.1 Работа с объектно-ориентированными возможностями по теме модуля

  1. Создайте переменную на основе new \DateTime() 
        $date = new DateTime();
  2. С помощью метода setTimestamp() установите количество секунд, прошедших от 1970 до 12 апреля 1961 
        $start = new DateTime("1970-01-01");
    $end = new DateTime("1961-12-04");
    $diff = $end->diff($start)->days * 86400;
  3. С помощью метода setTimezone установите часовой пояc DateTimeZone::EUROPE 
        $date -> setTimestamp($diff) -> setTimezone(new DateTimeZone('Europe/Moscow'));
  4. Отформатируйте дату через метод format('Y/m/d H:i:s') и выведите данные 
        echo $date -> format('Y/m/d H:i:s');

Результат (дата в формате 'Y/m/d H:i:s'): 1978/01/29 03:00:00

Лабораторная 1.2 Реализация автозагрузки

Реализуйте PSR-4 автозагрузку своих файлов через composer

Для демонстрации автозагрузки в этом файле подключается файл Cats, созданный на одном из занятий, содержащий класс Cat: 

    namespace Examples;
    require_once "../vendor/autoload.php";
    use DateTime;
    use DateTimeZone;

Пространство имен Examples определено для автозагрузки PSR-4 в composer.json: 

            "autoload": {
        "psr-4": {
            "Examples\\": "examples/"
        }
    },

В этом коде так же вызываются стандартные библиотеки php (DateTime и DateTimeZone). Это необходимо для корректной работы кода в части 1.1 лабораторной работы.

Далее создадим и напечатаем объект класса cats:

    $cat1 = new Cat('coco');
    echo $cat1 -> name;

Результат (параметр name созданного подгруженного через автозагрузку класса Cat): coco
